Как решу задачу разработки умного бота для поиска резюме
Архитектура решения: 
– модуль поиска резюме на всех доступных источниках 
– модуль фильтрации по критериям 
– модуль автоматической отправки результатов 
– модуль панели управления
Технические детали: 
– использование Python для разработки бота, 
– интеграция с API сайтов вакансий, 
– использование Selenium для парсинга сложных сайтов, 
– работа с ротирующими прокси для обхода блокировок, 
– интеграция с Telegram Bot API и email-сервисами, 
– использование LLM для умной сортировки кандидатов.

Мой опыт: 
Кейс 1: Парсинг сайтов застройщиков. Организовал разработку сервиса парсинга сайтов и фидов крупных застройщиков. Парсинг HTML-структур, извлечение данных. Опыт работы с парсингом сложных сайтов, обходом антибот-систем.
Кейс 2: Разработка мультиагентной системы. Разработал мультиагентную систему для риелторов, используя n8n, LangChain, MCP-серверы. Система включала интеграцию с различными API, обработку данных, работу с базами данных.

Ключевые моменты выполнения
Этапы работы: 
1. анализ требований и проектирование архитектуры бота, 
2. разработка модуля поиска резюме на всех доступных источниках с интеграцией API сайтов вакансий и парсингом Telegram-каналов, соцсетей, форумов, разработка модуля фильтрации по критериям (профессия, город и радиус, опыт работы), 
3. разработка модуля автоматической отправки результатов в Telegram или на email с настройкой уведомлений, 
4. разработка панели управления (веб или Telegram-меню) с умной сортировкой кандидатов по релевантности, интеграция всех модулей и тестирование бота, 
5. настройка обхода блокировок и капч. 

Оценка сроков и бюджета
Сроки: 21-28 дней (проектирование архитектуры и модуль поиска резюме 8-10, модуль фильтрации 3-4, модуль автоматической отправки результатов 3-4, модуль панели управления 4-6, интеграция и тестирование 3-4)
Бюджет: договорная (обсудим после уточнения деталей ТЗ и списка источников для поиска)
