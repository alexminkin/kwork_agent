Разработать в n8n телеграм-бота для создания нейрофотосессий.
Модуль «Проверка пользователя» в workflow телеграм-бота:
— Проверить, есть ли пользователь в базе данных; если отсутствует — добавить его.
Изначально в меню два пункта-кнопки на выбор:
1. «Сделать по текстовому промпту»
2. «Сделать по фото-референсу»
В любом варианте после выбора пользователь присылает свое фото (именно оно и является моделью для фотосессии)
Далее пользователь по запросу бота должен в первом случае - написать текстовый промпт-описание желаемой фотосессии, во втором случае - прислать фото желаемой оболочки фотосессии (может быть фон/образ/декор/украшения и тд)
Модуль «Генерация изображения» в ворфлоу ии-агента:
AI agent обрабатывает промпт и выбирает какая из нейросетей наиболее подходит под задачу.
Через API Kie.ai (это важно!) на выбор работают 3 нейросети:
1. Nanobanana - если по фото-референсу
2. Midjorney
3. seedream 4.0 - если текстовый промпт, фотография должна быть реалистичной
Модуль «Получение результата»:
Через бота пользователю отправляется готовое изображение от нейросети.
