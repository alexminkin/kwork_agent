Необходимо составить воркфлоу под ключ в Make или n8n.
Желаемая схема действия: Инструмент ищет площадки для размещения рекламы по ключевым словам/похожим сайтам -> На найденном сайте находит контакты для связи (предпочтительно адрес эл. почты либо форму для заполнения) -> Верифицирует адрес эл. почты -> Отправляет на этот адрес письмо с предложением о сотрудничестве. -> Через определенное время при отсутствии ответа отправляет на тот же адрес напоминание. -> Если есть ответ, он обрабатывается и вносится в таблицу. Все данные по ходу работы сценария заносятся в Google Sheets.
Первичные расходники в виде подписок на сервисы и API исполнитель задачи берет на себя, расходы включаются в конечную цену. У заказчика есть доступ к ChatGPT Plus.
