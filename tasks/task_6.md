Мы - маркетологи, не технические специалисты. Хотим создать чат-бота для Одноклассников, который будет помогать людям в тематических сообществах (огород, цветы, кулинария) советами по удобрениям. Бот должен стать полезным советчиком, а не спамером и параллельно продавать наши удобрение под брендом органопро.
Суть задачи:
Нужно разработать бота, который:
- Может отвечать на вопросы пользователей в комментариях и личных сообщениях в группах которые мы укажем
- Использует AI (YandexGPT или GigaChat) для генерации умных ответов
- Интегрируется с API Одноклассников
- Хранит историю диалогов
- Работает стабильно на сервере
Технические вопросы, которые нас волнуют:
- Возможно ли вообще сделать такого бота для Одноклассников без блокировки?
- Как лучше организовать архитектуру: Python + PostgreSQL + API Одноклассников + YandexGPT?
- Сможет ли бот адекватно работать в комментариях пабликов?
- Как обеспечить защиту от спама и ошибок?
- Что сложнее: настроить интеграцию с OK или сделать умный AI-модуль?
Мы предоставим:
- Контентную базу знаний по удобрениям
- API-ключи для AI-сервисов
- Тестовый аккаунт в Одноклассниках
Ищем исполнителя, который:
- Имеет опыт создания ботов для соцсетей
- Работал с API Одноклассников или ВК
- Знаком с YandexGPT/GigaChat API
- Может объяснить технические моменты простыми словами
