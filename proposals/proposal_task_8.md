Как решу задачу разработки телеграм-бота для нейрофотосессий

Архитектура решения в n8n: модуль проверки пользователя (проверка наличия в БД, добавление при отсутствии), модуль меню бота (две кнопки: Сделать по текстовому промпту, Сделать по фото-референсу), модуль обработки ввода (получение фото модели, обработка промпта или фото-референса), модуль генерации изображения (AI agent: анализ промпта и выбор нейросети, интеграция с API Kie.ai - Nanobanana для фото-референса, Midjourney, seedream 4.0 для текстового промпта), модуль получения результата (отправка готового изображения пользователю). Технические детали: использование n8n для создания workflow, интеграция с Telegram Bot API, работа с базой данных, интеграция с API Kie.ai для генерации изображений через три нейросети.
