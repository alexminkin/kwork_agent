Как решу задачу разработки ИИ-агента для создания анкет в резюме и бота для ответа в соцсетях

Архитектура решения: модуль создания анкет в резюме (интеграция с API площадок для размещения резюме, автоматическое создание анкет на основе шаблонов или данных пользователя, обработка различных форматов анкет для разных площадок), модуль бота для ответа в соцсети и Telegram (интеграция с API соцсетей и Telegram Bot API, обработка входящих сообщений через LLM, генерация ответов на основе контекста, автоматическая отправка ответов). Технические детали: использование Python для разработки системы, интеграция с API площадок для размещения резюме, использование LLM для генерации анкет и ответов, интеграция с API соцсетей и Telegram Bot API для работы бота.
