Как решу задачу разработки умного бота для поиска резюме

Архитектура решения: модуль поиска резюме на всех доступных источниках (интеграция с API сайтов вакансий HH, Avito, SuperJob, парсинг Telegram-каналов, соцсетей, форумов и прочих открытых площадок, использование Selenium для обхода антибот-систем и капч, работа с ротирующими прокси для стабильного обхода блокировок), модуль фильтрации по критериям (фильтрация по профессии, городу и радиусу, опыту работы, обработка описаний и ссылок на источники), модуль автоматической отправки результатов (интеграция с Telegram Bot API для отправки в бот или канал, интеграция с email-сервисами для отправки на email, настройка ежедневных или мгновенных уведомлений при появлении новых кандидатов), модуль панели управления (веб-панель или Telegram-меню для настройки фильтров и критериев поиска, умная сортировка кандидатов по релевантности, возможность масштабирования под другие профессии и города). Технические детали: использование Python для разработки бота, интеграция с API сайтов вакансий, использование Selenium для парсинга сложных сайтов, работа с ротирующими прокси для обхода блокировок, интеграция с Telegram Bot API и email-сервисами, использование LLM для умной сортировки кандидатов.
