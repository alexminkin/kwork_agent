# Коммерческое предложение: Агент n8n по сбору семантического ядра и кластеризации

## Опыт работы с n8n и парсингом

Опыт работы с n8n: разработал мультиагентную систему для риелторов, используя n8n для оркестрации процессов и тестирования. Опыт парсинга: организовал разработку сервиса парсинга сайтов и фидов крупных застройщиков (от Самолет и ФСК до региональных компаний), парсинг HTML-структур, извлечение мета-тегов и заголовков.

## Предложения по оптимизации архитектуры

**Архитектура воркфлоу:**
1. Модуль сбора семантики: параллельные запросы к SerpAPI/Google Custom Search для ускорения обработки
2. Обработка данных: batch-обработка результатов парсинга перед отправкой в OpenAI для снижения затрат на API
3. Кластеризация: использование embeddings для предварительной группировки запросов перед финальной кластеризацией через GPT
4. Дедупликация: сравнение через semantic similarity (embeddings) вместо точного текстового совпадения для более точного определения дублей
5. Кеширование: сохранение результатов парсинга доменов для исключения повторных запросов

**Обработка данных:**
- Асинхронная обработка доменов через n8n параллельные execution
- Валидация и очистка данных на этапе парсинга
- Структурирование данных перед записью в Google Sheets для оптимизации записи
