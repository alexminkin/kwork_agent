Архитектура воркфлоу для поиска площадок и автоматизации рассылки

Модули воркфлоу:
1. Поиск площадок: использование SerpAPI или Google Custom Search API для поиска сайтов по ключевым словам
2. Парсинг контактов: извлечение email-адресов и форм обратной связи с найденных сайтов через HTML-парсинг
3. Верификация email: проверка валидности email через API сервисов верификации
4. Отправка писем: интеграция с email-сервисами для отправки писем с предложением о сотрудничестве
5. Управление напоминаниями: отслеживание времени отправки и автоматическая отправка напоминаний при отсутствии ответа
6. Обработка ответов: мониторинг входящих писем через IMAP/POP3 или API, парсинг ответов
7. Интеграция с Google Sheets: запись всех данных (найденные площадки, контакты, статусы отправки, ответы) в таблицу через Google Sheets API

Обработка данных:
- Структурированное хранение данных о площадках и контактах
- Логирование всех действий для отслеживания прогресса
- Валидация данных перед записью в Google Sheets
- Обработка ошибок и повторные попытки при сбоях
